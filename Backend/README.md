# BFF.Web

Proyecto generado con .Net8 para el Backend.

## Requerimientos:

- Asegurar la separación de lógica en capas.
- Utilizar correctamente los patrones de diseño.
- Implementar conceptos SOLID.
- Implementar middleware para el manejo de errores.


## Levantar el server del api:

Corriendo el comando `dotnet run` el API se desplegará en [http://localhost:5013].

## Levantar el UI de swagger:

Levantado el server del API, swagger se encuentra en la ruta [http://localhost:5013/swagger].

## Correr los unit tests:

Corriendo el comando `dotnet test` ejecutará los unit tests via [xUnit].
